|Данный 2D проект использует архитектуру AMVCC|
-----------------------------------------------
*A - Aplication // Точка входа в программу
*M - Model //Модель для хранения данных, может содержать вычисления
*V - View //Эллемент пользовательского интерфейса, хранит данные и компоненты интерфейса
*C - Controller // Система управления и связывания View и Model
*C - Component // Любые скрипты, не относящиеся к остальным (Например скрипт вращения)
